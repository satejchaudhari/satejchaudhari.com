How to add a new writeup

Three steps, no build tools:

  1. Copy post-template.html, rename it, put it in /posts/, and write your content inside the post-body div.
  2. Open assets/js/posts.js and add one entry to the POSTS array at the top, pointing at your new file.
  3. Save and refresh index.html. That's it — it appears on the homepage.

Code blocks

Fenced code renders in a dark panel automatically:

import requests

def check_target(url):
    r = requests.get(url, timeout=5)
    return r.status_code == 200

Suggested tag values

Tags are free text, but a small consistent set keeps the log scannable — and any tag you use automatically gets its own filter button on the homepage:

  • WRITEUP — CTF or real-world exploitation walkthroughs
  • NOTES — shorter field notes, TIL-style
  • TOOL — tool reviews, scripts, tutorials
  • RESEARCH — deeper technical investigations
